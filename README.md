Blogger Website

- Blogger website similar to medium where user can post blogs
## Tech Stack

**Client:** React (Typescript), TailwindCSS

**Server:** Cloudflare Workers

**Authentication:** Zod, JWT, TypeInference

**Database:** Postgresql

**ORM:** Prisma
